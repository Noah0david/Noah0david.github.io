Mon projet sur les données

Cette page serra exclusivement consacré a mon projet.

  • Présentation générale de mon projet

    • Durant le confinement nous avons réaliser un projet complexe qui porte sur la création d'une carte interractive a partir de données publique. J'ai trouver ce projet vraiment très difficile contenu du temps que j'y ai passé, je n'ai pas réussi a crée de graphique pour mes données. J'ai eu beaucoup de problème avec les fichiers csv et ma bibliothèque csvreader que j'ai du modifié plusieurs fois car les codes n'étaient pas bon. Mais ce projet étais intéressant.

  • Présentation de la base de données publique utilisée

    • Ma base de donnée principale est une base de donnée contenant le nombres de test covid réalisé par jour dans chque departement avec le nombre de cas positif pour chque test. J'ai aussi utilisé une base de donnée contenant les localisations des hôtels de prefecture de chaque departement pour pouvoir avoir des points de repère sur ma carte.

  • Présentation de ma démarche

    • J'ai d'abord voulu réaliser une carte qui portais sur la couverture réseaux en france(2G/3G/4G) puis sur le déploiement de la fibre optique mais je n'ai pas trouvé de fichier intéréssant. J'ai donc fouillé un peu sur le thème de la météo puis les accident pour enfin finir sur les test au covid 19 en France. J'ai essayé de réaliser le contour des départements mais je n'y suis pas parvenue, j'ai donc décidé d'utiliser une table contenant la localistaion des préfectures de chaque département pour mettre un popup a chaque localisation. Une fois cela fait j'ai trié mes tables pour pouvoir faire des graphiques mais je n'ai pas réussis a les faires, j'ai donc crée des popup affichant les test effectué le premier jour où ont été entrée des données dans le fichier csv.
