Mes Projets de NSI

En NSI, nous avons réalisé plusieurs projets codés en Python, voici une liste de programmes sur lesquels nous avons travaillé avec des explications.

Le premier projet que je vais vous présenter est un programme en Python bien sûr qui permet de trier une liste dans l'ordre croissant. Ce n'est pas l'algorithme le plus rapide qui existe mais il est très pratique sur des petites listes. Vous trouverez ci-joint un pdf avec le code de cet algorithme. Tri par selection

Le second programme que je vais vous présenter est une interface réalisée avec le module et la bibliothèque Tkinter de Python. Cette interface n'est pas complète et ne permet pas de réaliser d'action particulière mais elle regroupe plusieurs éléments intéressants pour du codage graphique. Interface avec Tkinter

Pendant le confinement causé par le corona virus ;) nous avons étudié le chapitre sur les données et la cartographie. Nous avons appris a uttiliser des fichiers CSV en les manipulant avec Python. Notre projet final consiste a crée une carte interactive basée sur des données publique. Projet sur les Données
